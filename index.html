🚀 Instalação Rápida

Pré-requisitos

  • Python 3.11 ou superior instalado
  • Token de um bot Telegram (obtido com @BotFather)
  • Acesso a um terminal/linha de comando

Passo a Passo

1. Clone o repositório

git clone https://github.com/henriquetourinho/id-telegram-bot.git
cd id-telegram-bot

2. Configure o ambiente virtual (recomendado)

python -m venv venv
# No Linux/macOS:
source venv/bin/activate

# No Windows:
venv\Scripts\activate

3. Instale as dependências

pip install -r requirements.txt

4. Configure seu token

Abra o arquivo id_telegram_bot.py e substitua TOKEN pelo token do seu bot.

5. Execute o bot

python id_telegram_bot.py

📚 Como Usar

Descobrir seu ID

Simplesmente envie qualquer mensagem para o bot. Ele responderá com seu ID de usuário.

Identificar outros

Encaminhe uma mensagem de outro usuário, grupo ou canal que deseja identificar.

Privacidade

Se um usuário ocultou seu ID, o bot informará que não é possível identificá-lo.
